# GMV-video

<img width="1280" height="auto" alt="Game Movie icon" src="https://github.com/user-attachments/assets/740b079e-c737-45d3-a217-af52a4cdab28" />


A custom video file format for games, mostly for cutscenes.

This format is **NOT** for regular use, and is **SPECIFICALLY** for games. This isn't for complex animations too, since it can get big in file size... but you can play bad apple on it though if you want

## Contributing

This is 100% open-source! You can port it to ANY engine, and change the video format itself!

This project was made by a solo dev
